DIY AI Assistant: Step-by-Step Guide

Follow these steps to build your own private AI assistant from scratch. Requires intermediate Linux knowledge.

Flash JetPack OS & Configure Hardware

‚è±Ô∏è Estimated: 2-3 hours

Download NVIDIA JetPack 6.x from the official site. Flash to microSD using Etcher or dd. Boot the Jetson, complete initial Ubuntu setup. Install NVMe SSD and migrate the root filesystem from SD to NVMe for better performance:

  • Flash JetPack image to microSD
  • Boot and configure Ubuntu (user, password, network)
  • Partition NVMe: sudo parted /dev/nvme0n1
  • Copy root filesystem: sudo rsync -axHAWX / /mnt/nvme/
  • Update /boot/extlinux/extlinux.conf to boot from NVMe
  • Verify CUDA: nvcc --version

Install AI Runtime & Models

‚è±Ô∏è Estimated: 2-4 hours

Install Ollama for model management and download your first LLM. Configure TensorRT optimization for the Jetson GPU:

  • Install Ollama: curl -fsSL https://ollama.com/install.sh | sh
  • Pull a model: ollama pull llama3:8b-q4_0
  • Install Whisper for speech-to-text
  • Test inference: ollama run llama3
  • Troubleshoot CUDA/cuDNN version mismatches (common issue)
  • Optimize models with TensorRT (optional, +30% performance)

Install & Configure OpenClaw

‚è±Ô∏è Estimated: 3-6 hours

Clone OpenClaw, install dependencies, configure messaging platform connections, set up browser automation:

  • Install Node.js 20+, Python 3.11+, Docker
  • Clone: git clone https://github.com/openclaw/openclaw
  • Install dependencies: npm install
  • Configure .env with API keys, Telegram bot token, etc.
  • Set up Chrome for browser automation
  • Configure systemd service for auto-start on boot
  • Set up reverse proxy (optional, for remote access)
  • Debug connection issues, port conflicts, permissions...
